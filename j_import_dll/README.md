# j_import_dll
#### 概要
1. 引用dll中的方法，无需c/c++代码；只需要直接定义对应的结构体，然后引用即可。<br/>
使用方式相对比较稳定，对于dll的引用，很多项目都已经做的比较好了，比例:lxn/win; govcl等项目。<br/>
目前主要的dll: user32, shell32, kernel32, gdi32;
